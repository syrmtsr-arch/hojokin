# 石川県・北陸スタートアップ支援制度データベース（2026年版）

TENJOメンバー・入居者への紹介を目的に収集した補助金・支援制度一覧。

---

## 【補助金・助成金】（金銭的支援あり）

### 1. のとスタ補助金（能登へのスタートアップ誘致現地実証等支援事業費補助金）

| 項目 | 内容 |
|---|---|
| 主体 | 石川県（のとスタ） |
| 対象 | 能登地域の課題解決・新産業創出に取り組むスタートアップ |
| 補助額 | **最大1,000万円／年（最大3,000万円・3年）** |
| 補助率 | 4分の3以内 |
| 公募期間 | **2026年5月18日〜6月12日**（県公式ページで確認済み） |
| 支援内容 | 能登での実証事業＋専任プロデューサー伴走・デモデイ参加 |
| 備考 | 能登半島地震復興文脈。今回収集中の最大補助額制度 |

### 2. 新商品・新サービス開発支援事業助成金（旧・チャレンジ支援ファンド）

| 項目 | 内容 |
|---|---|
| 主体 | ISICO 新商品・サービス開発支援課 |
| 対象 | 石川県内に事業所を持つ中小企業（創業予定者も可） |
| 補助額 | 区分により異なる（例：上限300万円・補助率2/3） |
| 公募期間 | **2026年4月20日〜6月12日 16時必着** |
| 採択通知 | 2026年9月（予定） |
| 申請方法 | jGrants（電子申請）またはメール（shigen@isico.or.jp） |
| 問合せ | 076-267-5551 |

**メニュー区分**
- 1・2：地域資源活用型の新商品開発＋販路開拓
- 3・4：事業化可能性調査（FS）
- 5：海外販路拡大支援
- 6：大学・公設試等との連携開発
- 7・8：社会課題解決型（地域資源活用不要）
